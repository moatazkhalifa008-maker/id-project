# id check — النسخة الأولية (مع ربط الكاميرا)

## ما الذي يعمل الآن؟
- واجهة عربية بسيطة بــ Kivy
- التقاط صورة جديدة بالكاميرا عبر Plyer
- طلب صلاحية الكاميرا وقت التشغيل
- تحديد مجلد حفظ الصورة قبل الالتقاط
- استيراد صورة موجودة بمسار يدوي
- شاشة مراجعة للقراءة فقط
- منع التكرار بالرقم القومي
- البحث الجزئي بالاسم بعد التطبيع العربي
- تصدير CSV بالأعمدة: الاسم، السن، العنوان، الرقم القومي

## ملاحظات مهمة
- يستخدم التطبيق مجلدًا افتراضيًا داخل مساحة التطبيق إذا لم يُدخل المستخدم مسارًا.
- الـ OCR الحالي تجهيزي/Stub للاختبار فقط.
- لو ظهرت مشكلة على بعض الأجهزة الحديثة مع Intent الكاميرا، فالخطوة التالية ستكون تحسين الربط بطبقة Native أعمق.
